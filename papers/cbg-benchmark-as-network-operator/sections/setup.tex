% §5.1 Dataset characteristics:
%   - Public RIPE-based dataset: 713 anchors, three ASN fleets (AS16509/30 global VPs,
%     AS7018/125 US VPs, AS3209/164 EU VPs), 5-fold cross-validation.

%   - Proprietary operator dataset: user-plane/mobile-core RTTs, one target ASN,
%     200+ IPs across 20+ US clusters, VP topology matched to public dataset.
%
% §5.2 Data processing and setup construction:
%   - Target clustering: complete-linkage agglomerative, R=50 km, 257 global / 32 US / 21 DE cells.
%   - VP grouping by ASN + geographic scope.
%   - Geometry maps per setup (VP locations, centroid cells, fleet geometry diagnostics).
%   - K-fold: (K-1)/K anchor folds → LTD calibration; held-out fold → eval targets.
%
% §5.3 Validation primitive — Speed-of-internet intersection (calibration-free):
%   SoI circle: C_v = { x : dist(v,x) ≤ (2/3)c · RTT/2 }.
%   Physics-consistent iff claim ∈ ⋂_v C_v (unanimous; relax to majority for congested VPs).
%   Tradeoff: SoI-consistent ≠ uniquely identified (multiple candidate sites may be inside).
%   Bootstraps Anchored-track training from Geofeed/rDNS records.
